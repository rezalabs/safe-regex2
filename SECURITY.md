# Security

## Reporting a Vulnerability

If you discover a security vulnerability in safe-regex2, please report it by
opening an issue at https://github.com/rezalabs/safe-regex2/issues.

Please include:

- A description of the vulnerability
- Steps to reproduce
- The affected version(s)
- Any potential mitigations you've identified

Do not include sensitive details (tokens, credentials, etc.) in public issues.
If the vulnerability requires private disclosure, mention that in the issue and
we will provide a secure channel.

## Scope

This security policy covers the `@rezalabs/safe-regex2` npm package and its
source code in this repository.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 6.x     | :white_check_mark: |
| < 6.0   | :x:                |
