# Contributing

Bug reports and feature requests are welcome via [GitHub Issues](https://github.com/rezalabs/safe-regex2/issues).

## Pull Requests

This project does not accept pull requests. If you find a bug or have a feature suggestion, please open an issue instead.

## Development Setup

```sh
git clone https://github.com/rezalabs/safe-regex2.git
cd safe-regex2
npm install
```

## Testing

```sh
npm test
```

This runs the unit tests (`node --test`) and TypeScript type tests (`tstyche`).

## Linting

```sh
npm run lint
```

Uses [`neostandard`](https://github.com/neostandard/neostandard) with `ts: true`.
